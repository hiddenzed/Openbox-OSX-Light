Setups
=======
* **Panel** : `vala-panel`
* **Dock** : `plank` 
* **App Launcher** : `rofi`
* **Notification** : `dunst`
* **Compositor** : `compton`
* **Term Emulator** : `rxvt-unicode`
* **File Manager** : `pantheon-files`

Installation
==============

```
$ git clone https://github.com/noirecat/Openbox-OSX-Light.git
```

And just open **OSX-Light.obt** to apply the themes.
